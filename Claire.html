6-Week Progression Model
GYMGEEKS&C+ Programme Periodisation
Claire — Block Progression · Weeks 1–6
Load → Power → Peak  ·  Experienced Athlete Periodisation
Weeks 1–2 · Load
  • Establish SL RDL pattern (wall touch ok)
  • BSS: load at 7–8 RPE from day 1
  • SL box jump: bilateral takeoff → SL land
  • Skater bound: focus on stick, not distance
  • Pull-up: add weight if >6 clean reps
  • Note L/R asymmetries — flag if >10%
  • Copenhagen plank: short lever
Weeks 3–4 · Intensify
  • Remove all wall-touch aids
  • BSS: increase load 5–10%, drop to 6 reps
  • SL box jump: full SL takeoff + landing
  • SL RDL to Row: increase KB load
  • RFESS Jump: add if landing clean
  • Copenhagen: long lever + hip pulses
  • SL Hip Thrust: tempo eccentric 4s
Weeks 5–6 · Peak
  • BSS: heaviest load of block, 5 reps
  • Skater bound: max distance + stick
  • SL RDL to Row: pause at hinge
  • Weighted KB Dead Bug introduced
  • Test ride: uphill LBP comparison
  • Reassess L/R asymmetry scores
  • Plan next block: introduce trap bar DL
